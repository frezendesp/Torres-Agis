TORRE 2 — Servidores, File Servers e Datacenter

Responsável por toda camada de processamento, armazenamento e serviços de aplicação corporativos da AGIS, na Matriz, nas obras e no Datacenter SBA.

Missão

Missão da Torre

Garantir que todos os servidores, file servers e ambientes de Datacenter operem com disponibilidade, segurança, performance e capacidade adequadas para suportar o negócio, aplicações críticas, banco de dados e arquivos corporativos.

Esta torre é o núcleo de processamento da TI: tudo o que roda em servidores físicos ou virtuais passa por aqui — desde o Active Directory até as aplicações corporativas e o armazenamento de arquivos de obra.

Escopo

Escopo da Torre

Servidores na Matriz

  • Servidores Windows e Linux no CPD (6º andar)
  • Serviços internos de apoio (AD, utilitários, serviços locais)
  • Integração com switches core e firewalls Fortigate

Servidores de Obra

  • Servidores de obra para acesso a file server local
  • Ambientes de obras “encerradas mas ativas” para consulta
  • Sincronização/controladoria de dados com Datacenter

Datacenter SBA / Matrix

  • Servidores de aplicação corporativa
  • Servidores de banco de dados
  • Ambientes de homologação e desenvolvimento
  • Servicedesk, Zabbix e demais serviços corporativos
  • Storage corporativo e de obras finalizadas (OwnCloud)

File Servers e Armazenamento

  • File server da Matriz
  • File servers de obras
  • Storage de obras finalizadas (Owncloud / Datacenter)
  • Estrutura de pastas corporativas, departamentos e projetos
Camadas

Camadas de Serviços de Servidor

Camada de Diretório e Autenticação

  • Active Directory (domínio FGuedes)
  • Controladores de domínio distribuídos (Matriz, obras, SBA)
  • Integração com VPNs, WiFi e serviços internos

Camada de Aplicação

  • Servidores de aplicação Protheus, RM e sistemas de engenharia (no DC)
  • Servidores web internos e externos
  • Serviços de integração, APIs e middlewares

Camada de Banco de Dados

  • Servidores de banco (SQL Server, outros)
  • Instâncias de produção, homologação e desenvolvimento
  • Rotinas de manutenção (índices, integridade, jobs)

Camada de Arquivos

  • Shares de departamentos
  • Shares de projetos e obras
  • Estratégias de cotas, organização e segurança de pastas
Datacenter

Datacenter SBA / Matrix

O Datacenter é o backbone de processamento da AGIS. Nessa torre, a equipe coordena:

  • Relacionamento técnico com a Matrix / SBA
  • Gestão de capacidade (CPU, RAM, disco, IOPS)
  • Requisições de novos servidores ou expansão de recursos
  • Regras NAT, VPNs e rotas com a Matriz

Toda a parte de servidores corporativos principais (ERP, bancos, sistemas de apoio e monitoramento) está diretamente ligada a esta torre e é tratada em conjunto com a Torre 1 (redes) e Torre 6 (monitoramento).

Backup

Backup e Recuperação

Backup de Servidores

  • Backup Acronis no Datacenter SBA
  • Rotinas diárias de backup de aplicação, banco e arquivos
  • Backups full, diferenciais e incrementais conforme política

Recuperação e Testes

  • Testes de restore em ambientes de homologação
  • Restores pontuais para recuperação de arquivos e bases
  • Ajuste de janelas de backup conforme crescimento de dados
Operação

Operação Diária da Torre 2

  • Provisionar novos servidores (em conjunto com Matrix/SBA)
  • Ajustar e acompanhar uso de CPU, RAM, disco e rede dos servidores
  • Controlar espaço de file servers e storage
  • Atuar em incidentes de indisponibilidade de serviços
  • Manter inventário de servidores e serviços hospedados
  • Alinhar com times de sistemas/ERP sobre janelas de manutenção
Integrações

Integrações com Outras Torres

Com Torre 1 — Redes

  • Dependência total de VPNs, VLANs e rotas
  • Necessidade de baixa latência e banda adequada

Com Torre 3 — Suporte

  • Atendimentos de usuários em relação a file server
  • Demandas de espaço, permissões e lentidão

Com Torre 4 — Ativos

  • Inventário de servidores
  • Relação de sistemas e aplicações hospedadas

Com Torre 6 — Monitoramento

  • Monitoramento de disponibilidade de servidores
  • Alertas de CPU/RAM/disco/serviços críticos
